收集哪些資訊以及如何使用這些資訊

 

收集資訊的種類

 

為了向您提供我們的服務，我們會要求您提供該等服務所需的個人資訊。若您未提供您的個人資訊，我們可能無法提供我們的產品或服務。

 

我們只會就指定、明示及合法的用途，收集所需資訊，不會以有違此類用途的方式做進一步處理。我們可能會收集以下類型的資訊（無論是否為個人資訊）：

·        您提供給我們或上傳的資訊：我們可能會收集您提供給我們的所有個人資訊，例如您的姓名、手機號碼、電子郵件地址、身分證號碼、小米帳號詳細資料、與建立帳號及參與小米平台相關的資訊、您插入到聯絡人中或用來傳送訊息的電話號碼、意見反應，以及您提供給我們的任何其他資訊。

·        網路資訊：與您的網路狀態相關的資訊。包括網路 SSID、存取模式、閘道位址、WAN 連接埠上傳/下載速度、加密模式、Wi-Fi 通道、非純文字密碼、MU-MIMO 開關狀態、2.4G/5G 頻段開關狀態以及 VPN 組態資訊（若使用者已設定相關資訊）。

·        路由器資訊：與您的小米路由器 相關的資訊。包括路由器啟動狀態、綁定狀態、啟用狀態、共用狀態、機型、系統版本、MAC 位址、裝置序號、裝置 ID、國家/地區代碼以及路由器位置。

·        裝置資訊：連線至網路的裝置資訊。包括裝置連線狀態和計時、裝置名稱、IP 位址、MAC 位址、裝置類型、裝置品牌、連線類型、訊號強度、雜訊強度、上傳/下載速度和輸送量、最大上傳/下載速度、訪客 Wi-Fi ID 資訊、訪客網路連線和中斷時間、作業系統、裝置上線持續時間和 Wi-Fi 連線頻率；此外，當裝置超過某個閾值時，會報告頻寬使用率和使用持續時間。

·        位置資訊：與您的位置相關的資訊。包括國家/地區代碼、預設時區、預設日期、預設時間和預設語言。

·        硬體和系統資訊：LED 燈開關狀態。

·        功能設定資訊：我們可能會收集路由器功能設定的相關資訊，例如網路設定、定時重新啟動、密碼設定、IPTV、IGMP、QoS 速度限制、NFC、Wi-Fi 最佳化、設定定時器以停用 Wi-Fi、訪客 Wi-Fi、WPS 等。

·        登入資訊：與您使用特定功能、應用程式及網站有關的資訊。例如 Cookie 和其他匿名識別技術、IP 位址、網路要求資訊、臨時訊息歷史記錄、標準系統記錄、當機資訊。

·        其他資訊：我們可能也會收集其他類型的資訊，這些資訊與個人沒有直接或間接關聯，且為彙整性、匿名或去識別化的資訊。例如，使用特定服務時，可能會收集使用者的小米裝置的裝置型號和系統版本編號。我們收集此類資訊是為了改善為您所提供的服務。

·        意見反應資訊：當您在 App 中向我們提供問題或建議的意見反應時，我們可能會收集問題記錄、您主動提供的聯絡資訊及意見反應的內容，以上傳至伺服器。

 

如何使用個人資訊

 

收集個人資訊的目的，在於為您提供服務和/或產品，並遵守相關法律的規定。在此您同意我們處理您的個人資訊，並揭露該等資訊給我們的相關企業以及第三方服務供應商（如下定義），其用途如本隱私權政策所載。

 

我們可能使用您的個人資訊作為以下用途：

·        提供、處理、維護、改進及開發我們的商品和/或服務給您，包括售後服務及客戶支援，以及在您裝置上或透過我們網站提供的服務。

·        進行有關您的裝置、服務或任何一般問題的溝通，例如更新、客戶疑問支援、我們的活動資訊和通知。

·        分析並形成有關我們產品和服務使用的統計資訊，以更加提升我們產品與服務的品質。

·        最佳化您裝置的效能。

·        存留您的相關資訊，以支持我們的業務營運或履行法律相關責任。

·        無需與我們的伺服器通訊，直接提供本地服務。

 

以下列有更多詳細資料，用於說明我們如何使用您的資訊（其中可能包含個人資訊）：

·        設定您的小米帳號。在我們的網站上或透過我們的行動裝置建立小米帳號時所收集的個人資訊將用於為使用者建立個人小米帳號和個人資料頁面。

·        應用程式更新。為了讓您能夠持續享有小米 Wi-Fi 服務，我們將為您提供應用程式更新。在此功能期間，我們可能會收集路由器名稱和系統版本資訊，向您顯示裝置的版本資訊和功能更新。

·        評估和顯示網路狀態。我們會讀取下載速度、上傳速度、網路延遲、行動網路安全狀態、Wi-Fi 通道狀態、雜訊強度、通道輸送量、下載任務狀態、上傳狀態和訊號強度等資訊。這些資訊將用於評估和顯示 Wi-Fi 品質、下載狀態、上傳狀態以及訊號強度。請放心，這些資訊僅會在本機讀取，不會儲存在雲端。

·        為裝置提供網路連線服務。為了向裝置提供網路連線服務，我們可能會收集裝置連線狀態和計時、裝置名稱、IP 位址、MAC 位址、裝置類型、裝置品牌、連線類型、訊號強度、雜訊強度、上傳/下載速度和輸送量、最大上傳/下載速度、訪客 Wi-Fi ID 資訊、訪客網路連線和中斷時間、作業系統、裝置上線持續時間和 Wi-Fi 連線頻率。

·        連線至伺服器。收集國家和地區的相關資訊以區分適合的伺服器，讓使用者可以正常享受小米路由器 服務。

·        Wi-Fi 自動重新連線。為了使用 Wi-Fi 自動重新連線功能來連線至您小米帳號下的 IoT 裝置，我們需要查詢您的使用者 ID 以驗證裝置的身分，進而確保安全性。

·        服務品質 (QoS)。我們提供 QoS 功能。當您開啟 QoS 功能時，我們可能會收集頻寬資訊、上傳/下載速度、您設定的速度限制類型（例如遊戲優先順序）以及您設定的上傳/下載速度限制等資訊。這些資訊將用於智慧速度分佈和速度限制。

·        常規訪客 Wi-Fi。我們支援為訪客設定專門的 Wi-Fi 子網路。若您設定了訪客 Wi-Fi，所支援的裝置將以訪客身分連線至子網路，進而將其與主網路區分開來。為了啟用此功能，我們可能會收集您設定的功能開關狀態、網路 SSID、加密狀態、加密密碼和網路終端資訊。這些資訊將用於提供訪客 Wi-Fi 和網路裝置顯示功能。

·        Wi-Fi 定時器，排程重新啟動。為了提供更完善的小米 Wi-Fi 服務，我們支援設定 Wi-Fi 定時器和排程重新啟動。當您啟用這些功能時，我們可能會收集您的 Wi-Fi 重新啟動時間，以便您設定 Wi-Fi 定時器和/或重新啟動。

·        傳送通知。我們可能會不時使用您的個人資訊傳送重要通知，例如有關購買的通訊以及條款、條件和政策的異動說明。

·        收集使用者意見反應。您提供的意見反應對於我們改善服務十分寶貴。為了對您選擇提供的意見反應進行後續追蹤，我們可能會使用您提供的個人資訊與您保持聯絡並儲存相關記錄以上傳至伺服器。

安全保護措施

 

小米的安全措施

 

我們承諾確保您個人資訊的安全。為了防止未經授權的存取、揭露或其他類似風險，我們採取合理的實體、電子和管理程式，來維護和保障我們從您使用小米產品與服務中以及在小米網站上所收集的資訊。我們將竭盡所有合理努力來保護您的個人資訊。

 

例如：當您存取小米帳號時，可選擇使用我們的雙步驟驗證流程，以提高安全性。當您將資料從小米裝置傳送或接收至我們的伺服器時，我們確保這些資料會使用安全通訊端層（以下稱「SSL」）和其他演算法進行加密。

 

您所有的個人資訊均儲存在由受控制設施保護的安全伺服器中。我們會依據重要性及敏感性來區分您的資料，並確保您的個人資訊受到最高安全層級的保護。我們確保存取資訊以協助您使用產品與服務的員工和第三方都經簽約嚴格遵守機密責任之約束，且若未能履行責任，可能遭受處罰或解雇。對於雲端的資料儲存我們也設有特別的存取控管。從各方面來說，我們經常審查資訊收集、儲存和處理的做法，包括實體性的安全措施來保護資料不受任何非授權存取和使用。

 

我們將採取所有可行手段來保護您的個人資訊。但您應瞭解，使用網際網路並非絕對安全，因此我們無法保證透過網際網路由您轉移或轉移予您的任何個人資訊之安全性或完整性。

 

您可採取的行動

 

·        您不應向任何人揭露您的登入密碼或帳號資訊，除非其已獲得您的恰當授權，以確實保護您的個人資訊。不論何時在小米網站上以小米帳號使用者身分登入（特別是在其他人的電腦或公用網際網路終端機上），請務必在工作階段結束時登出。

·        倘若因您未能將個人資訊妥善保密，導致第三方存取您的個人資訊從而造成的安全漏洞，小米概不負責。儘管有上述規定，若有任何其他網際網路使用者未經授權使用您的帳號，或出現任何其他違反安全措施的情況，您仍須立即通知我們。

·        您的配合將有助我們保護您個人資料的隱私權。

 

保留政策

 

只要履行收集個人資訊之目的所需，或根據適用法律的要求或允許，我們將會留存相關的個人資訊。我們一旦合理認為收集個人資訊之目的不必再經由留存個人資訊來履行，我們即會停止留存個人資訊，或取消可將個人資訊與特定個人相關聯的方式。倘若是基於記錄大眾利益目的、科學或歷史研究目的，或是根據適用法律的統計資料目的進行進一步處理，則即使進一步處理不符合原始用途，小米仍然會進一步儲存資料。

 

在您將裝置重新設定至出廠預設值時，裝置上所有的個人資訊均會被清除。若要執行強制重設功能，請按下重設按鈕約 5 秒。當系統狀態指示燈變為橘色，然後路由器再次完成啟動時，裝置即成功執行原廠預設設定。請注意硬重設無法復原，因此您必須確認您已不再需要這些資訊。

 

存取您裝置上的其他功能

 

我們的應用程式可能需存取裝置上的某些功能，例如 Wi-Fi 網路狀態。此資訊用於允許應用程式在您的裝置上執行，並允許您與應用程式進行互動。您隨時可在裝置層級關閉這些權限，或透過 https://privacy.mi.com/support 與我們聯絡來撤銷您的許可。

控制隱私權設定

 

小米瞭解對於隱私權的疑慮將會因人而異。因此，我們提供了一些範例來說明小米提供的方式，您可選擇用於限制個人資訊的收集、使用、揭露或處理，並可控制您的隱私權設定：
